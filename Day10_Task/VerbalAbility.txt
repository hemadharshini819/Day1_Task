Reading Passage

Binary search is an efficient searching algorithm that works on sorted arrays.
It repeatedly divides the search interval into half and reduces the search space.

Questions

1. Where does binary search work?
SOL: It is an efficient searching algorithm that works on sorted arrays.

2. How does binary search reduce search space?
SOL: It repeatedly divides the search interval into half and reduces the search space.

3. Is binary search faster than linear search?
SOL: Yes.Binary search is faster because it halves the data each step (O(log n)),
while linear search checks every element(O(n)).
