VERBAL ABILITY (Advanced)

Reading Passage:

Operating systems manage hardware resources and provide services for computer programs.
Efficient OS design ensures better performance, memory management, and process scheduling.

Questions

1. What is the main role of an operating system?
ANS: To manage hardware resources and provide essential services

2. Why is memory management important?
ANS: It is a core component of efficient OS design, which ensures
better overall system performance

3. What is process scheduling?
ANS: A method used by the OS to manage how and when different programs
and access the CPU to ensure smooth multitasking.
